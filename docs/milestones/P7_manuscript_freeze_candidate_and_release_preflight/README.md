# P7 Manuscript Freeze Candidate and Release Preflight

Goal: turn the completed `P6` manuscript bundle into a clear freeze candidate
and outward release-preflight package without reopening scientific scope.

Scope:

- define freeze-candidate criteria for the current manuscript bundle;
- fix the final intended main-text figure/table ordering and appendix-companion
  inclusion plan;
- prepare an outward release-preflight checklist for README / STATUS / release
  summary / paper-facing ledgers;
- keep any future blog work explicitly downstream and blocked until those
  conditions are met.

Non-goals:

- no new experiments by default;
- no reopening of `R1`, `R2`, or `D0` evidence without a separate evidence wave;
- no blog drafting ahead of release-preflight clearance.

Working package:

- `docs/publication_record/freeze_candidate_criteria.md` defines the minimum
  meaningful freeze threshold;
- `docs/publication_record/main_text_order.md` fixes first appearance order for
  the main-text figure/table set;
- `docs/publication_record/appendix_companion_scope.md` classifies required
  versus optional companions;
- `docs/publication_record/release_preflight_checklist.md` defines the outward
  release gate;
- `docs/publication_record/blog_release_rules.md` keeps blog work downstream;
- `handoff.md` records the later parallel execution order.
