# Acceptance

This milestone is counted as passed only for the current narrow real-trace
claim:

- real extracted streams must be evaluated, not only synthetic address scans;
- horizon inflation must be an explicit variable;
- at least one native-pass real memory stream must flip under horizon inflation
  in the single-head float32 scheme;
- at least one decomposed float32 scheme must stay stable where single-head
  fails on the same stream;
- the suite must include more than one real program family.

The milestone is still not a proof of broad long-horizon robustness.
