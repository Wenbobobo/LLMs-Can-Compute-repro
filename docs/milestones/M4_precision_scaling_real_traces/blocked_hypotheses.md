# Blocked Hypotheses

- Synthetic local stability alone was not enough evidence for real-trace
  stability.
- Passing at a stream's native horizon is not enough evidence of robust
  temporal precision; the new sweep shows that horizon inflation can flip some
  single-head results from pass to fail.
- The current real-trace suite suggests that decomposition helps materially,
  but it is still too narrow to support any serious long-horizon marketing
  claim.
