read-only inventory of the dirty root quarantine after `P85` rebaselined live
control onto merged `main`.

branch as `wip/root-main-parking-2026-03-24`, classifies dirty paths into
`duplicate_or_obsolete`, `salvage_candidate`, and `archive_only`, and marks the
