# E1 Conditional Evidence Reopen Protocol

Goal: define the only allowed route for reopening evidence after the current
post-`P7` stabilization package.

Scope:

- enumerate valid reopen triggers;
- define bounded patch-lane classes;
- require every reopen to return to the locked paper/release lanes.

Non-goals:

- no active evidence work by default;
- no open-ended widening of precision, systems, or compiled scope.
