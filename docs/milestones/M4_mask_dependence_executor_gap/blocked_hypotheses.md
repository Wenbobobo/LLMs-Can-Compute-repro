# Blocked Hypotheses

- “The staged branch is already equivalent to a general neural executor.”
  Blocked because held-out structural rollout is still `0.0`.
- “`opcode_shape` is already enough, we only need more training.”
  Blocked until the current `opcode_shape` failures are classified by field and
  family.
- “Another stronger regime should be added immediately.”
  Blocked until the existing failures show a narrow enough missing-constraint
  pattern to justify one more regime.
