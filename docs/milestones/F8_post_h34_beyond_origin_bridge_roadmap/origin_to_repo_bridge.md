# Origin To Repo Bridge

This note maps "beyond origin" ambitions onto the current repo without
relabeling them as current results.

`planning_only` below means docs-only classification or comparator work, not an
authorized runtime or implementation lane.

| Origin-facing ambition | Current repo anchor | Current status | Nearest admissible next form |
| --- | --- | --- | --- |
| Deterministic computation from append-only traces | `H28/R34/R35/H29` Origin-core packet | `supported_here` | preserve as the current scientific floor |
| Narrow compiled exactness on the same substrate | `R37/R38/R39/H32/H34` | `supported_here` | preserve as narrow compiled-boundary support only |

The bridge value of `F8` is therefore classificatory, not operational. It keeps
future families visible without implying that any of them are currently active.
