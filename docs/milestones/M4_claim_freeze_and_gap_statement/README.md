Goal: freeze one canonical post-M4 claim statement after mask-dependence and
precision-generalization close.

Canonical freeze:
- supported with caveat: the staged pointer branch (`C2f`) closes exact rollout
  on the current toy suite only under the strongest `opcode_legal` regime;
- negative closure: the broadened `M4-D` suite (`C2h`) does not justify a fair
  fourth regime, because cleaned failures split between `push_expr_0`
  memory-value mismatches and `step_budget` nontermination;
- sharpened boundary: the broadened precision suite (`C3e`) shows immediate
  float32 single-head failure on all new high-address memory families, while
  current decomposition schemes remain stable on the exported suite.

Blocked broadening:
- do not restate this as “LLMs are computers”;
- do not restate this as a fair learned executor result independent of decode
  legality;
- do not open compiled demos beyond the tiny typed-bytecode boundary until this
  claim freeze is the active narrative source.
