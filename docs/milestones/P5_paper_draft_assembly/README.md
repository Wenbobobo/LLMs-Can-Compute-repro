# P5 Paper Draft Assembly

Goal: turn the now-frozen claim set into a paper-ready manuscript skeleton
without reopening scientific scope.

Scope:

- expand the paper outline from a topic list into a sectioned manuscript plan;
- map each section to the exact evidence paths already frozen by `P3`, `R1`,
  `R2`, `M7`, and `P4`;
- define what belongs in main text versus appendix, so later prose work does
  not silently widen the claim set.

Non-goals:

- no new experiments by default;
- no new frontend widening;
- no blog-first narrative that outruns the paper bundle.
