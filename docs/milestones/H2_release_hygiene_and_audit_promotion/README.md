# H2 Release Hygiene and Audit Promotion

Goal: promote the post-`P7` bundle-lock and release-hygiene rules into a
machine-audited standing gate for unattended work.

Scope:

- define post-`P7` submission and release control docs;
- add one new bundle-lock audit with matching tests and result artifacts;
- update the public-surface audit wording to track the new stage;
- record the new standing gate in the experiment manifest.

Non-goals:

- no new scientific evidence;
- no reopening of precision, systems, or compiled scope;
- no broader outward release.
