# P26 Acceptance

- `commit_split_manifest.md` recommends a reviewable packet split relative to
  dirty `main`;
- `main_delta_summary.md` records the current branch relationship without
  relabeling it as merged;
- `artifact_tracking_policy.md` classifies the user-mentioned large raw file
  as outside the current clean source bundle unless it is actually present on
  the source branch;
- `worktree_runbook.md` records a future promotion procedure without
  authorizing it now.
