# Acceptance

This milestone is complete only if it narrows or strengthens the precision
claim with explicit evidence.

Minimum evidence requirements:
- at least three real-trace families beyond the current easy offset core;
- per-stream records of native horizon, first-failure multiplier, and failure
  type;
- a stable failure taxonomy that distinguishes at least `tie-collapse`,
  `wrong-address inversion`, and `other`;
- claim language updated to exactly the suite that survives.

This milestone is **not** complete if it only adds more pass/fail rows without
explaining the failure modes.
