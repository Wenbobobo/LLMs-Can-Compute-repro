# Blocked Hypotheses

- “The current offset suite is already broad enough.”
  Blocked because it still over-represents transformed loop-style traces.
- “More decomposition families should be added next.”
  Blocked until the current three schemes are mapped across a broader set of
  real streams.
- “Passing at native horizon is enough evidence of robustness.”
  Blocked because horizon inflation has already produced counterexamples.
