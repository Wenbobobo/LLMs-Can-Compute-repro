# P1 Paper Readiness

Goal: preserve enough structure that future paper drafting does not depend on
memory or reruns.

This milestone is evidence-discipline, not prose-writing.

Frozen responsibilities:
- keep `claim_ladder`, `claim_evidence_table`, `negative_results`,
  `experiment_manifest`, `threats_to_validity`, and `figure_backlog`
  synchronized after every unattended batch;
- define the mandatory figures and tables needed for a first paper draft;
- keep appendix-level diagnostics attached to the same claim row when they do
  not justify a new claim layer;
- preserve enough metadata that each major claim can be traced back to one
  concrete artifact bundle.
