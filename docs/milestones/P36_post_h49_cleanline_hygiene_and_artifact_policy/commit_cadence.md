# P36 Commit Cadence

- commit `F26` planning surfaces separately from all runtime work;
- commit `P36` hygiene surfaces separately from scientific claims;
- commit `R51` runtime outputs separately from `R52` comparator/value outputs;
- commit `H50` decision surfaces only after `R51/R52` are explicit; and
- do not mix packet closeout, runtime execution, and merge-posture changes in a
  single commit.
