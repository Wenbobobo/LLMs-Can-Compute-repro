# H1 Legacy Backlog Reconciliation

Goal: reconcile historical unchecked milestone rows against the current frozen
paper scope so unattended runs do not treat legacy backlog as active work.

Scope:

- classify every currently unchecked legacy row under one explicit disposition;
- annotate affected historical `todo.md` files with a pointer to the
  classification matrix;
- record which rows are frozen, superseded, or only conditionally reopenable.

Non-goals:

- no new experiments;
- no reopening of frozen evidence waves by default;
- no public-facing claim change.
