## Goal

Convert the completed `P7` freeze-candidate checkpoint into a post-`P7`
submission/release stabilization package without reopening scientific scope.

## Fixed Inputs

- `freeze_candidate_criteria.md`
- `main_text_order.md`
- `appendix_companion_scope.md`
- `release_preflight_checklist.md`
- `blog_release_rules.md`
- `submission_candidate_criteria.md`
- `release_candidate_checklist.md`

## Package Write Set

- `P8`: manuscript bundle, appendix package, and paper-facing ledgers;
- `H2`: audits, release hygiene, and manifest/accounting;
- `P9`: README, STATUS, release summary, and blocked-blog derivative controls.

## Exit Gate

The package exits only when:

1. the submission-candidate bundle is locked;
2. `P1`, `P5`, and `H2` audits are green;
3. the restrained public surface is synchronized downstream of the locked
   bundle;
4. no `E1` patch lane is active.
